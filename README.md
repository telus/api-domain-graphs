# API Domain Graphs

This repo stores the DOT source for Telus Digital's API Domain graphs.

Change the graph source, run the build, update slides with the new images.

## Getting Started

### Install Graphviz

```bash
  brew install graphviz
```

### Vector build

```bash
  ./svg_build.sh
```

### Raster build

```bash
  ./png_build.sh
```
